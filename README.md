# Curvature-Driven-Catalysis

Official implementation of the paper **"A Geometric-Electronic Principle for Curvature-Driven Single-Atom Catalysis"**.

This repository provides the complete model-generation and post-processing workflow used in the study, including curvature-controlled surface construction, automated orbital identification, wavefunction analysis, and the orbital centroid (⟨z⟩) calculation.

## Repository structure

```
Curvature-Driven-Catalysis/
├── README.md
├── LICENSE                        # MIT License

│   └── dz2_dn/                   # Exported dz² probability densities
├── templates/
│   ├── INCAR_relax               # Ionic relaxation (Fe/N free, C fixed)
│   ├── INCAR_scf                 # Static SCF (LWAVE=.TRUE. for orbital export)
│   └── KPOINTS                   # Gamma-only
└── src/
    ├── unifyGenerate.py           # Curvature-controlled FeN4/SAC structure generation
    ├── vasp_analyzer.py           # PDOS peak identification + WAVECAR export
    ├── compute_z_centroid.py      # ⟨z⟩ centroid calculation
    ├── pca_orbital_analysis.py    # Orbital density PCA decomposition
    └── io_utils.py                # Shared CHGCAR I/O utilities
```

## Installation

```bash
git clone https://github.com/Migie-cphy/Curvature-Driven-Catalysis.git
cd Curvature-Driven-Catalysis
pip install -r requirements.txt
```

`vasp_analyzer.py` additionally requires [**vaspwfc**](https://github.com/QijingZheng/VaspBandUnfolding) for WAVECAR reading.

## Workflow overview

The analysis pipeline consists of four stages:

```
1. Structure generation       2. DFT calculation       3. Orbital extraction       4. Post-processing
   unifyGenerate.py     -->      VASP (external)  -->   vasp_analyzer.py     -->   compute_z_centroid.py
                                                                              -->   pca_orbital_analysis.py
```

### Step 1 — Generate curvature-controlled SAC structures

`unifyGenerate.py` constructs FeN4-embedded graphene sheets with systematically varied principal curvatures (κ₁, κ₂) and orientation angle θ.

```bash
cd src
python unifyGenerate.py
```

This generates VASP POSCAR files in `../data/fix_SAC/` with naming convention `{k1}_{k2}_{theta}.vasp`.

### Step 2 — DFT calculation

Run spin-polarised DFT with VASP for each generated structure. Input templates are provided in `templates/`:

1. **Relaxation** — copy `INCAR_relax` as `INCAR`. Carbon atoms are fixed via selective dynamics; Fe and N are relaxed (`ISIF=2`).
2. **Static SCF** — after relaxation, copy `CONTCAR` to `POSCAR` and use `INCAR_scf` as `INCAR`. This generates `WAVECAR` (`LWAVE=.TRUE.`) and PDOS data (`LORBIT=11`) needed for orbital export.

Key settings: RPBE (`GGA=RP`), DFT+U on Fe 3d (U_eff = 2 eV), DFT-D3 dispersion (`IVDW=12`), VASPsol implicit solvation. Gamma-only k-point sampling.

`PDOS_USER.dat` is generated by [VASPKIT](https://vaspkit.com/) (option 115) from the static SCF results.

### Step 3 — Orbital identification and wavefunction export

`vasp_analyzer.py` automatically identifies the dominant d-orbital peak from PDOS data, locates the corresponding band in the OUTCAR eigenvalue list, and exports its real-space probability density |ψ|² from the WAVECAR.

```bash
cd /path/to/vasp/calculation
python /path/to/src/vasp_analyzer.py
```

**Inputs**: `OUTCAR`, `DOSCAR`, `OSZICAR`, `PDOS_USER.dat`, `WAVECAR`, `POSCAR` (all in the working directory).

**Output**: `prob_density_s{spin}_b{band}_r.vasp` — a CHGCAR-format file containing |ψ|².

### Step 4a — Compute ⟨z⟩ centroid

`compute_z_centroid.py` reads the exported `.vasp` files (CHGCAR-format probability densities), crops an integration box centred on the cell centre, and computes the z-direction density centroid. The z origin is at the geometric centre of the box. Both the original density (`expectation_p`) and z-flipped density (`expectation_m`) are computed — the latter serves as a self-consistency check (`_m ≈ −_p`).

```bash
cd src

# Default: process all .vasp files in data/dz2_dn/
python compute_z_centroid.py

# Custom pattern and output
python compute_z_centroid.py -p '../data/dz2_dn/*.vasp' -o z_centroid.csv

# Custom integration box size (Å)
python compute_z_centroid.py -p '*.vasp' --box 5.0 5.0 8.0
```

**Output**: `z_expectation.csv` with columns `structure_id`, `expectation_p`, `expectation_m`.

### Step 4b — PCA orbital shape analysis

`pca_orbital_analysis.py` performs PCA on the set of cropped orbital densities to identify the dominant shape variations across the curvature series.

```bash
cd src

# Default: process all .vasp files in data/dz2_dn/
python pca_orbital_analysis.py

# Custom pattern and box
python pca_orbital_analysis.py -p '../data/dz2_dn/*.vasp' --box 4.0 4.0 6.0
```

**Outputs**:
- `pca.csv` — PCA projections for each structure
- `intensity.csv` — integrated density per structure
- `pca_variance_ratio.png` — scree plot
- `pca_projection_2d.png` — 2-D PC1 vs PC2 projection
- `pca_results/` — CHGCAR-format mean density and principal components (viewable in VESTA)

## Data availability

Large data files (WAVECAR, DOSCAR, exported probability densities `*.vasp`, etc.) are excluded from this repository due to GitHub size limits. The complete dataset is archived on Zenodo:

> [![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18823122.svg)](https://doi.org/10.5281/zenodo.18823122)

After downloading, extract the archive into the repository root so that the `data/` directory is populated:

```bash
unzip data.zip
```
